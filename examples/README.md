# Pawly Examples

- `basic_usage.py`: local heuristic path and optional cloud-provider path if `pawly-cloud` is installed
- `run_actions_basic.py`: recommended one-call decision + execution path using `SkillRegistry` and top-level `run_actions(...)`
- `run_actions.md` in `../docs/`: recommended one-call decision + execution flow using `SkillRegistry`
- `execution_gateway_demo.py`: execution wrapping demo
- `agents/`: example worker declarations
