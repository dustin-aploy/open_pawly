# Certification Logic

The Pawly test suite treats certification as a repeatable evidence process around Pawprint and Pawly behavior rather than a self-assertion.

A certified result requires:

- use of the packaged Pawprint schema and internal report formats;
- execution of the full local check catalog;
- zero failed checks; and
- a generated compliance report that captures the run outcome.

This repository provides the local foundation for that process. Review and ranking may happen elsewhere, but they do not replace the suite itself.
