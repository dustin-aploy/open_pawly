# Paw Adapters

The public adapter implementation now lives in `src/pawly/adapters`.

Use:

- `pawly.OpenAIAgentsPawAdapter`
- `pawly.ClaudeSkillsPawAdapter`
- `pawly.CrewAIPawAdapter`
- `pawly.LangGraphPawAdapter`
- `pawly.OpenClawPawAdapter`
- `pawly.SelfHostedHTTPAdapter`

This top-level `adapters/` directory is compatibility guidance only. Do not add or maintain a second adapter implementation here.
